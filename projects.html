Projects

A collection of my personal tech projects

Sarcasm-Detection

    This project implements a sarcasm detection model for news headlines using a sequence-based neural network. The dataset, sourced from professional news sites (The Onion for sarcasm and Huffington Post for regular news), ensures high-quality labels and minimal noise. Utilizing TensorFlow, Keras, and pre-trained embeddings, the final CNN-based model achieves an accuracy of 99.15% on the test set, making it highly effective for identifying sarcastic headlines.

Stock-prediction-RNN

    This project utilizes Recurrent Neural Networks (RNNs) with LSTM layers to predict stock prices based on historical time series data. Leveraging Python, Keras, and NumPy, the model learns from past stock price movements and generates predictions for future values. Through feature scaling and careful data preprocessing, the trained model achieves a reasonably low error (RMSE), providing a useful insight into stock trend forecasting.

Bicep_curl_Buddy

    This Python application leverages OpenCV and MediaPipe to provide real-time pose estimation and rep counting during bicep curls. By tracking key landmarks (shoulder, elbow, and wrist), it dynamically measures angle changes to identify movement stages and accurately count repetitions. The result is a valuable tool for users looking to monitor their exercise form and progress.

Text-Prediction-using-RNN

    This project trains a Recurrent Neural Network (RNN) using LSTM layers to generate text predictions based on Irish song lyrics. By learning word patterns and sequences from the input dataset, the model can produce original, poetry-like lines. Leveraging tokenization, padded sequences, and word embeddings, it captures linguistic structures and styles from the source text, resulting in coherent yet creatively generated verses.

movie-recommender

    This Streamlit-powered movie recommendation application suggests similar titles based on a selected movie. Leveraging a precomputed similarity matrix (trained on movie metadata, cast, crew, and keywords), it retrieves and displays the top recommended films alongside their posters. The front-end is designed with Streamlit, while data and model artifacts are loaded from pickle files and queried through an API for posters, offering users a smooth and interactive movie discovery experience.
